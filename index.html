üìö How to Find Your SharePoint Library ID

Follow these simple steps to get your SharePoint Library ID parameters:

1 Open SharePoint Library
Navigate to your SharePoint library in your web browser (Chrome, Edge, Firefox, etc.).
2 Click the Sync Button
Look for the Sync button in the toolbar (usually near the top of the page). Click it to start the sync process.
3 Wait for Dialog Box
A dialog box will appear saying "We are syncing your files" or similar message. Don't close this dialog!
4 Copy Library ID
In the sync dialog box, look for a link or button that says "Copy library ID" or similar. Click it to copy the encoded parameters to your clipboard.
5 Paste into Formatter
Return to the URL Formatter and paste the Library ID into the input box. Click Convert to get the formatted version!
üí° Pro Tip: The copied Library ID contains all the parameters you need (tenantId, siteId, webId, listId, etc.). Just paste it directly into the formatter - no need to edit anything first!
